            <ul>
                <li>
                    <h4>Separación de Contenido y Estilo:</h4>

                </li>
            </ul>
            <ul>
                <li>
                    <h4>Diseño Responsivo:</h4>
                    Con CSS, puedes crear diseños que se adapten a diferentes tamaños de pantalla, lo que es esencial para <br> la creación de sitios web responsivos que funcionen bien en dispositivos móviles, tabletas y <br> computadoras de escritorio.
                </li>
            </ul>
            <ul>
                <li>
                    <h4>Consistencia Visual:</h4>
                    CSS proporciona un conjunto de reglas y estilos que se pueden aplicar de manera consistente a través <br> de un sitio web completo. Esto garantiza una apariencia visual coherente y profesional.
                </li>
            </ul>

            <ul>
